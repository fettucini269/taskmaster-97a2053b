# TaskMaster

A simple task management application to help users organize their daily tasks.

## Features

- Add new tasks with deadlines
- Mark tasks as completed
- View all tasks
- Edit existing tasks
- Delete tasks

## Tech Stack

- Python Flask
- PostgreSQL
- Docker

## Environment Variables

- `DATABASE_URL` - PostgreSQL connection URL (alternative to individual vars below)
- `DB_HOST` - Database host
- `DB_PORT` - Database port (default: 5432)
- `DB_NAME` - Database name
- `DB_USER` - Database user
- `DB_PASSWORD` - Database password

## Running Locally

```bash
# Install dependencies
pip install -r requirements.txt

# Set environment variables
export DB_HOST=localhost
export DB_NAME=myapp
export DB_USER=postgres
export DB_PASSWORD=postgres

# Run the application
python app.py
```

## Docker

```bash
# Build image
docker build -t taskmaster .

# Run container
docker run -p 8080:8080 -e DATABASE_URL=postgresql://user:pass@host/db taskmaster
```

## API Endpoints

- `GET /` - API info
- `GET /health` - Health check
- `GET /items` - List all items
- `POST /items` - Create new item
- `GET /items/<id>` - Get item by ID
- `PUT /items/<id>` - Update item
- `DELETE /items/<id>` - Delete item
